# Statistics

Legacy ASP.NET-проект на C#/.NET Framework для статистических расчетов и отображения табличных данных.

## Назначение

Проект содержит ASP.NET приложение `Statistics` и связанные классы расчетов/таблиц. В исходниках присутствуют модели таблиц, расчетный класс `Calculate.cs`, стандартный ASP.NET MVC/Auth scaffolding и файл `Code.txt`.

## Состав

- `Statistics.sln` - решение Visual Studio.
- `Statistics` - ASP.NET-приложение.
- `ModelingProject1` - дополнительный проект/модель из исходного состояния.
- `Statistics/Classes` - расчетные классы и табличные модели.
- `Code.txt` - вспомогательный текстовый артефакт проекта.
- `packages`, `TestResults` - legacy NuGet/test artifacts.

## Технологии

- C#
- ASP.NET MVC
- .NET Framework
- Legacy Visual Studio tooling

## Сборка

Открыть `Statistics.sln` в Visual Studio с поддержкой ASP.NET для .NET Framework и восстановить NuGet-пакеты.

## Статус

Архивный учебный проект. Уже есть GitHub-репозиторий `MaximillianVoss/statistics`, который в текущем governance-проходе помечен как archived legacy repository.
